<!-- 
To check all ease options available visit:
https://gsap.com/docs/v3/Eases/

Group:
scroll, hover (reverse the anim on out), click.

Ins and Outs:
the variables from Ins is the value that the element will come from, and the Outs is the value that the element will go to.
For example, opacity of 0 for Ins means that the element will start with opacity 0 and go to the value set inside webflow, and the opposite for Outs.

Position and Rotation:
Don't set those as negative vales, the code will handle that for you.

Directions:
up, down, left, right.

Split Into:
lines, words, letters, elements (works on elements with children).

Timeline Position:
https://gsap.com/docs/v3/GSAP/gsap.timeline()/#positioning-animations-in-a-timeline

By default all timelines start at the same time, if you want to start at a different time you can set the position in the timeline.
-->
